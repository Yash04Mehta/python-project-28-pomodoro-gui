# Pomodoro Timer (Python Tkinter)

A desktop Pomodoro Timer built using Python and Tkinter that helps manage focused work sessions with automatic short and long breaks.

## Features

* 25-minute work sessions
* 5-minute short breaks
* 20-minute long breaks after every 4 work sessions
* Countdown timer with live updates
* Session progress tracking using checkmarks
* Reset button to restart timer anytime
* Simple tomato-themed UI

## Tech Stack

* Python
* Tkinter
* Math module

## How It Works

The timer cycles through:

1. Work session
2. Short break
3. Work session
4. Short break
5. Work session
6. Short break
7. Work session
8. Long break

Checkmarks appear after each completed work session.
